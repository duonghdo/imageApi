## imageProcessingApi
Resize an image to requested dimension

### Scripts
- Install: ```npm install```
- Build: ```npm run build```
- Lint: ```npm run lint```
- Prettier: ```npm run prettier```
- Run unit tests: ```npm run test```
- Start server: ```npm run start```

### Example
http://localhost:3001/api/images?filename=fjord&width=200&height=200
